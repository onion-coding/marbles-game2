# Code of Conduct

Internal contributors and external partners on the Marbles Game project
agree to interact under the following baseline:

1. **Respect** — disagree on technical choices, never on people.
2. **No harassment** — sexual attention, slurs, threats, doxxing,
   sustained disruption are all out, regardless of target.
3. **Confidentiality** — pre-release builds, customer (operator) data,
   and integration drafts stay with the people who need to see them.
   Don't post screenshots of unreleased builds or confidential operator
   discussions to public channels.
4. **Reporting** — issues with another contributor's behaviour go to
   `conduct@onion-coding.example` (set this up before adding external
   contributors). Reports are read by humans, not by the person being
   reported.
5. **Enforcement** — verified violations escalate from warning →
   temporary ban from the repository → termination of contributor /
   contractor agreement, depending on severity.

This is a working document; if it's missing a case you ran into, open a
PR.
